# 🧠 MONAI Deep Learning Medical Imaging Projects

This repository contains multiple deep learning projects built using **MONAI, PyTorch, and Medical Imaging Datasets**. Each project is maintained in a separate folder with its own full documentation.

---

## 📂 Projects in This Repository

### 🔹 1. Brain Tumor Segmentation – BRaTS Model
A deep learning pipeline for **3D Brain Tumor Segmentation** using the **BRaTS (Brain Tumor Segmentation) dataset** and **MONAI framework**.

✅ Features:
- 3D U-Net architecture
- Multi-modal MRI support (FLAIR, T1, T1CE, T2)
- Dice Loss optimization
- GPU-based training
- Full preprocessing & postprocessing pipeline

📁 Folder:
brats-model/


📄 Detailed Documentation:
👉 [View BRaTS Model README](./brats-model/Readme.md)

---

### 🔹 2. Retinal Disease Classification Model
A deep learning model for **retinal disease classification** using **fundus images** and convolutional neural networks.

✅ Features:
- Image classification using CNN
- Data augmentation
- Performance evaluation metrics
- Medical image preprocessing
- Optimized training pipeline

📁 Folder:
retinal-model/


📄 Detailed Documentation:
👉 [View Retinal Model README](./retinal-model/README.md)
